# Darts-Game

This is a school project. So don't expect too much.

## Prerequisites

-   [SAS](http://www.dingemann.de/sas/)


## Installation

First, install SAS in the maven local repo:
```bash
mvn install:install-file -Dfile="path\sas.jar" -DgroupId=sas -DartifactId=sas -Dversion=5.8 -Dpackaging=jar
```
